# 测试计划

## 1. 测试目标

验证 CPClaw 是否满足产品设计蓝图和技术设计蓝图中的核心目标：用户可以通过对话式入口，基于本地云枢元数据知识图谱，安全、可解释、可审计地查询和操作云枢应用。

## 2. 单元测试

### 2.1 Agent

- 意图识别。
- 槽位抽取。
- 本地元数据能力匹配。
- 非向量多路召回与候选排序。
- 多轮上下文指代解析。
- DAG 执行计划生成。
- Reflection 判断。
- 高风险操作拦截。
- 记忆召回和写入规则。

### 2.2 模型网关

- OpenAI 兼容请求构造。
- 模型配置读取。
- API Key 运行时解密。
- 思考模式能力校验。
- 结构化输出解析。
- 模型调用摘要脱敏。

### 2.3 安全

- 凭据加密和解密。
- 日志脱敏。
- 工具输入输出脱敏。
- Prompt 敏感信息过滤。
- 附件解析结果脱敏。
- `.env` 不入库检查。

### 2.4 云枢连接器

- 登录参数构造。
- Token/Cookie 缓存和脱敏。
- 元数据标准化。
- 应用级知识图谱构建。
- 查询参数构造。
- 写操作确认前拦截。

### 2.5 附件

- 文件类型白名单。
- 文件大小限制。
- 文件名规范化。
- 文本、表格、OCR 解析结果结构化。
- 字段映射生成。

## 3. 集成测试

- MySQL 连接。
- Spring Data JPA / Flyway 迁移。
- 会话保存和读取。
- 系统设置保存和读取。
- 模型配置保存、读取和测试。
- 云枢连接测试。
- 云枢架构同步。
- MySQL 元数据图谱写入。
- MySQL Metadata Index 写入和非向量检索。
- 后续 Elasticsearch/OpenSearch 索引写入和检索。
- 附件上传、解析和字段映射。
- 审计记录写入。

## 4. E2E 测试

使用 Playwright 测试：

1. 打开前端。
2. 进入设置页。
3. 填写云枢连接信息。
4. 配置模型 API 地址、API Key 和模型名称。
5. 配置模型是否支持思考模式。
6. 测试模型连接。
7. 测试云枢连接。
8. 同步云枢架构。
9. 新建会话。
10. 在对话框选择模型并启用或关闭思考模式。
11. 输入“查询我的销售订单数据”。
12. 检查返回摘要、Markdown 表格和执行步骤。
13. 输入“给第一条写一条跟进记录”。
14. 检查系统能解析上下文并要求确认。
15. 上传发票附件并要求填写报销单。
16. 检查字段映射、缺失字段提示和附件上传确认。
17. 查看历史会话和审计记录。

## 5. 安全验收

- Git diff 中没有明文密码、Token、Cookie、Session ID、API Key。
- 日志中敏感字段脱敏。
- 设置接口不返回密码、API Key 或密文内容。
- 模型调用摘要不包含敏感凭据。
- 附件解析结果不泄露敏感原文。
- 删除、批量修改、流程提交、业务 Action、附件上传均需要确认。
- 未确认或确认过期的高风险操作不得执行。
- `.env`、本地 Claude 配置、浏览器状态、日志和密钥文件不得入库。

## 6. MVP 验收标准

- Vue 3 前端可启动。
- Spring Boot 后端可启动。
- MySQL 配置可连接。
- 会话和消息可保存。
- 设置可保存，敏感凭据加密。
- 模型配置可保存并用于对话。
- 对话框支持模型选择和思考模式开关。
- 云枢元数据可同步到本地 MySQL。
- Agent 检索只访问本地 Metadata Index。
- Agent 可完成查询类模拟链路。
- 高风险操作确认机制可阻断未确认执行。
- 附件上传和解析链路具备基础能力。
- 文档按“产品设计蓝图、技术设计蓝图、专项详细设计、测试用例”分层存储在 `docs/`。
